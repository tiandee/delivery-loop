# Delivery Loop

An Agent Skill for carrying a software change from implementation to evidence-based delivery.

It helps an agent select an appropriate verification mode, preserve authorization boundaries, require fresh evidence, and validate the path that real users consume.

## Install

```bash
npx skills add https://github.com/tiandee/delivery-loop --skill delivery-loop -g
```

## Use

Ask your coding agent to use `delivery-loop` when a change must be implemented, verified, released, and validated end to end.
